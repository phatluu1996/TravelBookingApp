How to Start the Project?


How to run your react project please check the structure below:

Step 1 - go to /path-to-dir/dirto-react folder/directory

Step 2 - Open the Terminal or Command line in the dirto-react

Step 3 - Then run this command npm install in your terminal

Step 4 - After install then run this command npm start

Step 5 - Open http://localhost:3000 to view it in the browser.

How to add your image?


You can replace the placeholder images with your real images very easily. Please follow the steps below.

Step 1 - go to /path-to-dir/dirto-react/src/assets/images folder

Step - 2 - Replace any images you want with the same name. You can also add new images here and add this image in any component from this folder

How To Edit Components?


We have organized all components in a directory so that you can find and edit them easily. Please follow the step below:

Step 1 - go to /path-to-dir/dirto-react/src/components folder

Here you will be able to see all the components we have used. You can change any component or add new components very easily

Besides, we have added all pages in the /path-to-dir/dirto-react/src/pagesDirectory. You can edit any pages there easily.

How to css files?


You can edit your custom CSS very easily. Please follow the instruction below to edit your CSS.

Step 1 - go to /path-to-dir/dirto-react/public/css/style.css file. This is our main file.

Step 2 - go to /path-to-dir/dirto-react/src/App.css. You can add your custom CSS here. We have added this App.css file for custom CSS.
